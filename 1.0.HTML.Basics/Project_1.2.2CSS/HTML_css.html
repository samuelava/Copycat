Frågor och svar om CSS

Vad är CSS? Hur används det?

CSS står för; cascading style sheet, används för stilisera HTML-kod och styr utseendet på webbsidor.

Vad är skillnaden mellan inline-stil, inbäddad stil och extern stil i CSS?

Inline-stil är när man skriver CSS-kod inuti ett tag i HTML-kod, inbäddad-stil CSS är skriven inuti HTML dokumentet och extern-stil finns i ett separat dokument från HTML-koden.

Vad är en selektor i CSS och hur används den?

Selektorer i CSS används för att välja och tillämpa stilar på specifika HTML-element. Det finns fem olika sorters selektorer, enkla, kombinerade, pseudo-klass, pseudoelement, attribut.

Vad är skillnaden mellan klass-selektor och ID-selektor i CSS?

ID bör bara finns en gång per dokument, men klass kan finnas flera gånger.

Vad är kaskadordningen i CSS och hur påverkar den hur stilar tillämpas på element?

Arv, relevans, ursprung och betydelse, noggrannhet, ordning.

Vad är responsiv webbdesign och hur kan CSS användas för att skapa responsiva layouter?

Ändrar på layouten beroende på skärmstorlek. Samt kan man som användare av en webbplats ändra på hur det ser ut på enheten med hjälp av CSS.

Vad är boxmodellen i CSS och vilka element påverkas av den?

Det är en layout som innehåller content, padding, border och margin.

Vad är "grid layout" och hur skiljer det sig från "flexbox"?

Grid layout innehåller både kolumner och rader, medan flexbox bara har rader.

Vad är skillnaden mellan positionering med "static", "relative", "absolute" och "fixed" i CSS?

Static är standard och ändras inte av andra regler, relative likt static men kan ändras av andra regler, fixed följer med även när man scrollar på sidan och absolute positionerar sig närmast till föräldrar elementet istället för webbläsarfönstret.
