Frågor och svar om webborientering

Vad är en webbläsare och vad gör den?

Mjukvara som används för att integrera med webbsidor.

Vad är en webbserver och vad gör den?

En dator eller soft vara som bevarar och skickar webbsidor och andra web resurser.

Vad är en URL och vad innehåller den?

Uniform Resource Locator. Adresser för webbplatser. En URL innehåller protokoll prefix som HTTPS och sedan domännamnet som Google.com och ibland en mer specifik förklaring för domänet.

Vad är skillnaden mellan HTTP och HTTPS?

HTTP skickar data mellan webbläsare och webbservern, medan HTTPS krypterar datan för mer skydd från obehörig åtkomst.

Vilka är några exempel på frontend-ramverk och vad används de till?

Exempel är HTML för struktur, CSS för utseende, JavaScript för interaktivitet och det används för att skapa responsiva och snygga webbsidor.
