Frågor och svar om HTML

Vad är HTML och vad används det för?

HTML är en grundläggande byggsten för webben och är skyldig för strukturen av hemsidor.

Beskriv strukturen i ett HTML-dokument.

Doctype deklarerar vilken version av html som används, html innehåller all information om hemsidan, head där meta taggarna finns samt länkar till externa resurser, body innehåller all visuell information på hemsidan.

Vad är ett element? Ge exempel.

Definierar objekt och är strukturen av HTML Exempel på det kan vara:

  • < html lang=”sv”>
  • < meta charset=”utf-8”>

Vilka är de viktigaste skillnaderna mellan HTML5 och HTML4?

HTML5 är det mycket lättare att designa responsiva webbsidor.

Vad är XML och XHTML?

XML är likt HTML men det är mer flexibelt och XHTML är lite tvärtom och är mer strikt.
