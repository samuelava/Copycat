Frågor och svar om Java Skript

Vad är JavaScript och varför är det viktigt för webbutveckling?

Skapades 1995 och gör hemsidor mer interaktiva så att de inte bara är statiska, JavaScript kan också bevara information.

Vad är ECMAScript och hur förhåller det sig till JavaScript?

ECMAScript kommer ifrån ECMA-262 som är baserat på JavaScript.

Vad är variabler i JavaScript och hur deklarerar du dem?

Variabler kan vara textsträngar, nummer, boolean-värden och objekt. LET eller CONST, där LET är variabler där man kan ändra på deras värde och CONST har ett konstant värde. Några exempel på hur man deklarerar:

  • let name = “namn”
  • let namn = true;

Vad är en funktion i JavaScript och hur definierar du en?

En funktion definieras:

function < function-name>(arg1, arg2,...) { //write function code here };

Vad är "strict mode" i JavaScript och varför kan det vara användbart?

Identifierar felaktig kod, varnar och stoppar felaktiga användningar av variabler. Ibland kan den också öka prestanda.

Vad är skillnaden mellan "vanliga" JavaScript-bibliotek och ramverk som React, Angular och Vue?

Javascript-bibliotek fokuserar på enstaka funktioner, styckvis integration och är lätt att lära. Medans JavaScript-ramverk är svårare att lära sig och fokuserar på arkitektur på webbsidor.
